#### 开发思路

```
批量拖拽图片
批量生成表单
提交到后台
一条条处理
任何一条成功

前端支持一键提交

后台直接起多个线程同时执行保存
任何一个失败实时返回
```

#### 如何在 idea 创建一个新的项目
- 新建项目 选择spring boot
- copy 一个 配置文件 application.yml
- 创建对应的数据库
- 创建实体类 并 启动，如何 数据库 里自动生成了表，项目创建成功
